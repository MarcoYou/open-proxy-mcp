# 주주총회 안건 매뉴얼

## 안건 유형별 파서 매핑

| 안건 유형 | XML 파서 | 일반적 제목 패턴 | 비고 |
|----------|---------|---------------|------|
| 재무제표 승인 | parse_financials_xml | "재무제표", "대차대조표", "손익계산서" | 보고사항일 수 있음 (결의 안건 아닌 경우) |
| 이사/감사 선임 | parse_personnel_xml | "선임", "해임" | 집중투표 시 별도 처리 필요 |
| 정관변경 | parse_aoi_xml | "정관" | 하위 안건(제2-1호 등)으로 분할 빈번 |
| 보수한도 | parse_compensation_xml | "보수", "한도" | 이사/감사 분리 가능 |
| 자기주식 | parse_treasury_share_xml | "자기주식", "보유", "처분", "소각" | **XML 제목 매칭 한계**, PDF fallback 빈번 |
| 자본준비금 | parse_capital_reserve_xml | "자본준비금", "이익잉여금 전입" | 감액배당 전제 조건 (확정 아님) |
| 퇴직금 규정 | parse_retirement_pay_xml | "퇴직금", "퇴직급여" | 재무제표 주석의 "퇴직급여"와 혼동 주의 |

## 파싱 한계 및 주의사항

### 자기주식 (treasury_share)
- XML 파서는 안건 제목에 "자기주식/보유/처분/소각" 키워드로 매칭
- 소집공고에 자사주 안건이 명시적으로 없는 기업이 많음
- 정관변경 안에 자사주 관련 조항이 포함되거나, "자기주식" 대신 다른 표현 사용
- **PDF fallback이 정상 동작** — 본문 전체 스캔으로 잡음
- KOSPI 200 중 149/199 성공 (대부분 PDF fallback 경유)

### 보수한도 (compensation)
- "보수" 또는 "한도" 키워드로 안건 매칭
- 이사/감사 보수가 별도 안건일 수 있음 (제5호 이사, 제6호 감사)
- 금액 단위 주의: DART 원문은 "백만원", "천원" 등 다양 → limitAmount(원 단위)로 정규화
- 프론트엔드에서 formatKrwShort()로 "70억원" 형태 표시

### 퇴직금 규정 (retirement_pay)
- 재무제표 주석의 "퇴직급여" 테이블과 혼동 위험 → 본문 fallback 제거로 해결
- 안건 제목에 "퇴직금" 있을 때만 매칭

### 재무제표 (financials)
- **보고사항**일 수 있음 — 이 경우 결의 안건이 아니라 투표 자체가 없음
- 참석률 역산 시 보고사항 안건은 제외해야 함
- BS(재무상태표) + IS(손익계산서) 필수. BS만 있는 경우 IS 자동 감지 (매출/영업이익/순이자 계정)

### 정관변경 (aoi_change)
- 하위 안건(제2-1호, 제2-2호 등)으로 분할되는 경우 빈번
- "------생략" 표기는 정상 (변경 없는 부분 생략)
- "<삭제>"도 정상
- 집중투표제 배제/도입 관련 정관변경은 별도 의미 (2027년 의무화 대비)

### 이사/감사 선임 (personnel)
- 후보자 이름: 한글 2-5자, 영문 병기 시 더 길 수 있음
- 경력: 기간+내용 분리. 100자+ 한 줄 병합 시 PDF fallback
- 감사위원 선임: 최대주주 의결권 3% 제한 → 참석률 역산 시 분모 다름
- **집중투표**: 찬성/반대 없음, 득표율+순위만. 상위 N명 가결.

## 집중투표제

### 일반 투표 vs 집중투표
| | 일반 투표 | 집중투표 |
|--|---------|---------|
| 의결권 | 안건당 1주 1표 | N명 선출 시 1주 N표 |
| 결과 | 찬성/반대/기권 (%) | 득표율 + 순위 |
| 가결 기준 | 과반수 또는 2/3 | 득표 상위 N명 |
| DART 표기 | 발행기준/행사기준 찬성률 | "-" (찬성률 없음) |
| remark | - | "득표율: 16.8% 순위: 2위" |

### KOSPI 200 집중투표 실시 기업 (2026)
- 고려아연: 이사 5인 선임 (MBK vs 영풍 프록시 파이트)
- 금호석유화학

### 2027년 변화
- 대형상장사 집중투표제 의무화 예정
- 정관에 집중투표 배제 조항 삭제하는 기업 증가 (삼성전자, LG화학 등)

## 주총결과 (KIND 크롤링)

### 데이터 소스
- KRX KIND (kind.krx.co.kr) 크롤링
- DART rcept_no → KIND acptno 변환: 8번째 이후 "80" → "00"
- 3단계: 메인 페이지 → searchContents → 본문 HTML

### 섹션 구조 (XBRL 서식, 표준화)
| 섹션 | 출현율 | 내용 |
|------|--------|------|
| 정기주주총회 결과 | 100% | 재무제표+배당+임원현황 |
| 주주총회 안건 세부내역 | 100% | 투표 결과 테이블 |
| 사외이사선임 세부내역 | 90% | 후보자 경력 |
| 감사위원선임 세부내역 | 85% | 후보자 경력 |
| 이사선임 세부내역 | 76% | 사내이사 |
| 집중투표제 도입/배제 | 62% | 배제→적용 변경 등 |

### 참석률 역산
```
전체 참석률 = 발행기준 찬성률 / 행사기준 찬성률
```
- 보통결의 안건 중 최빈값이 대표 참석률
- 감사위원 선임은 3% 의결권 제한으로 분모 변경 → 참석률 달라짐
- 정확한 행사 주식수는 DART에 미공시 → 추정치

## 시간순서 규칙

```
소집결의 (이사회) → 소집공고 → 주총 당일 → 주총결과
```

- **공고 데이터를 결과에서 참조** → OK
- **결과 데이터를 공고에 넣기** → 금지 (시간 역전)
- v4 JSON은 **최종 통합본** — 공고+결과를 합친 것이지, 공고 시점에 결과를 넣은 게 아님
