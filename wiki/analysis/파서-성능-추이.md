## 핵심 개선 기법

- **bs4 HTML 직접 파싱**: text regex에서 HTML 구조 활용으로 전환
- **lxml 파서**: html.parser 대비 30% 속도 향상
- **경력 분리**: `<p>` 태그, 現/前 구분자, 연도 토큰 할당
- **PDF fallback**: [[opendataloader]] `table_method="cluster"`. [[XML-vs-PDF]] 분석에서 보강 전략 확정
- **OCR fallback**: [[Upstage-OCR]] Document Parse, 11건 100% 성공
- **침범 방지**: 본문 fallback 제거로 다른 안건 데이터 오인 최소화

## 남은 한계

- retirement_pay OCR 86.7%: 유일하게 OCR에서도 100% 미달
- treasury_share XML 93.6%: 소집공고에 명시적 안건 없는 기업 많음
