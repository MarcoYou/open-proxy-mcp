# corp_gov_report 설계

거버넌스 종합 평가를 위한 기업지배구조보고서 파서 tool. 15 → 16번째 tool, Data Tools 11개째.

## 동기

기존 OPM은 **개별 공시 분석**에는 강하지만 **거버넌스 종합 평가**가 빈 자리였음:
- 주총/지분/배당/분쟁 개별 tool → 낱낱의 사건 해석
- 거버넌스 총평 → 없음

`기업지배구조보고서`는 한국거래소가 정한 표준 서식으로, 15개 핵심지표 준수 여부 + 세부원칙별 응답 + 기업개요를 담음. OPM의 `corp_gov_report`가 이를 파싱해 종합 평가 데이터를 제공.

## 의무 범위 — 연도별 확대 일정 (정정됨 2026-04-22)

KRX가 자산 규모에 따라 단계적으로 확대. **2026년 제출분부터 KOSPI 전체 의무.**

| 제출 시기 | 대상 자산 |
|----------|----------|
| 2019년~ | 자산 2조원 이상 |
| 2022년~ | 자산 1조원 이상 |
| 2024년~ | 자산 **5천억** 이상 (2023 사업연도 기준) |

- **KOSDAQ**: 자율공시 (일부 대형 KOSDAQ 상장사만 자발 제출)

- **제출 시한**: 매년 **5월말**
- **연중 [기재정정]** 재제출 빈번 — 파싱 시 최신 건 우선

※ 이전 이 페이지는 "2024 사업연도부터 전체 KOSPI 의무"로 잘못 기재되어 있었음 (2026-04-22 정정).

## 소스 전략

전용 DART API 없음. 따라서:
1. `list.json` (pblntf_ty=I) + 키워드 `기업지배구조보고서`로 최근 4년 보고서 리스트
2. 최신 보고서의 원문(`get_document_cached`) 다운로드
3. BeautifulSoup으로 텍스트 추출 후 정규식·섹션 패턴 파싱

## scope

| scope | 내용 |
|-------|------|
| `summary` (기본) | 기업개요 + 준수율 + 15지표 ✅/❌ 요약 |
| `metrics` | 15지표 당기·직전기 O/X + 100자 비고 전수 |
| `principles` | 세부원칙별 응답 텍스트 (최대 30건) |
| `filings` | 최근 4년 제출 이력 |

## 파싱 핵심 패턴

### 기업개요 (표 1-0-0)
```
최대주주
현대모비스 외 9명
최대주주등의 지분율
29.68
소액주주 지분율
52.53
```
→ `label` + 다음 줄이 값. `_after(label)` 헬퍼로 매칭.

### 15 지표 표
```
핵심지표
(공시대상기간)준수여부
(직전 공시대상기간)준수여부
비고
주주총회 4주 전에 소집공고 실시
O
해당없음
’24년 정기주주총회시 개최일 34일 전에 송집공고 실시함
전자투표 실시
O
해당없음
...
```
→ 헤더 "비고" 이후 4줄씩 반복 (지표명 / 당기 / 직전기 / 비고)

### 세부원칙 응답
```
[원칙 설명 텍스트]
상기 세부원칙에 대한 준수여부를 간략하게 기술한다. (100자 이내)
당사는 주주총회 관련 정보를 소집통지서 및 전자공시시스템을 통해 충분한 기간 전에 제공하고 있습니다.
```
→ 정규식으로 "상기 세부원칙에 대한 준수여부를" 바로 다음 줄 추출

## 15개 핵심지표 (2024 기준)

### 주주 (5)
1. 주주총회 4주 전에 소집공고 실시
2. 전자투표 실시
3. 주주총회의 집중일 이외 개최
4. 현금 배당관련 예측가능성 제공
5. 배당정책 및 배당실시 계획을 연 1회 이상 주주에게 통지

### 이사회 (6)
6. 최고경영자 승계정책 마련 및 운영
7. 위험관리 등 내부통제정책 마련 및 운영
8. 사외이사가 이사회 의장인지 여부
9. 집중투표제 채택
10. 기업가치 훼손 또는 주주권익 침해 방지 정책
11. 이사회 구성원 모두 단일성(性)이 아님

### 감사기구 (4)
12. 독립적인 내부감사부서 설치
13. 내부감사기구에 회계 또는 재무 전문가 존재
14. 분기별 1회 이상 경영진 참석 없이 외부감사인과 회의
15. 경영 관련 중요정보에 내부감사기구 접근 절차 마련

## 파서 개선 (v1 → v2)

**v1 문제**: 4줄 고정 패턴(지표명/당기/직전/비고) 가정 → 비고 없는 서식(삼성, SK하이닉스 일부)에서 파싱 실패.

**v2 해결**: 15개 표준 지표 라벨 prefix(25자)로 위치 찾고, 각 블록에서 O/X 값 2개(당기·직전) + 나머지를 비고로 동적 수집. 비고 0개~다수 모두 대응.

또한 키워드를 `"기업지배구조보고서"` → `"기업지배구조보고서공시"` 엄격화 + `"연차보고서"` 명시 제외로 금융지주 별도 서식 필터링.

## 개선 방향 (진행 + 남음)

1. ✅ **다양한 원문 서식 대응**: 삼성·SK하이닉스 파서 보강 완료 (15/15)
2. ✅ **timeline scope 추가**: 연도별 준수율 + 지표 전환(improved / regressed / changed) 감지
3. ✅ **연차보고서 필터**: 금융지주 "연차보고서" 별도 서식 skip
4. **세부원칙 파싱 정교화** (미완): 현재 원칙명 매칭 단순, 누락 다수
5. **오래된 보고서 서식** (미완): 2022/2023년 현대차 보고서 준수율 파싱 실패 — 구 서식 추가 대응 필요
6. **ESG 평가등급 연동** (미완): KCGS 평가등급, 서스틴베스트 외부 데이터 결합

## timeline scope 동작

- 최근 5개 filings(rcept_no)을 순차 조회 → 각 원문 파싱 → 연도별 준수율 + 지표값
- `transitions` 블록: 지표별 O→X(regressed) / X→O(improved) / 기타 변화 자동 감지
- 현대차 실측: 2025/2024 보고서 80% 동일, 전환 이벤트 0건 (지표 안정)

## 거버넌스 분석에서의 의의

- 개별 의결권 행사 판단(`prepare_vote_brief`)에 거시 컨텍스트 제공
- 연도별 준수율 후퇴 = 거버넌스 리스크 조기 경보
- 동종 업계 기준선 비교 → 상대적 평가 가능
- 준수 O 이지만 100자 응답이 모호하면 형식적 준수 의심
