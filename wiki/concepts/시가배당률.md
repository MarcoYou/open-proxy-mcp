# 시가배당률

## 개념

[[DART-OpenAPI]]가 공식 공시하는 배당수익률. 일반적인 [[배당수익률]](종가 기준)과 계산 방식이 다름.

```
시가배당률(%) = 1주당 배당금 / 기준주가 x 100
기준주가 = 배당기준일 전전거래일부터 과거 1주일간 종가의 산술평균
```

## 일반 배당수익률과의 차이

| 항목 | 시가배당률 (DART) | 배당수익률 (자체) |
|------|-------------------|-------------------|
| 기준 가격 | 1주일 평균 종가 | 1일 종가 |
| 기준일 | 배당기준일 전전거래일 기준 | 배당기준일 |
| 소스 | DART alotMatter | 네이버 금융/KRX |
| 변동성 | 평활화 (평균) | 단일일 가격 |

## OPM에서의 우선순위

DART alotMatter와 현금배당결정 공시에 시가배당률이 있으면 해당 값을 우선 사용. 없을 때만 [[네이버-금융]] 종가로 자체 계산. [[div-tool-rule]]에서 우선순위 규칙 정의.
