# 특수관계인

## 개념

[[최대주주]]와 혈연, 혼인, 계열회사 등으로 연결된 자. 최대주주의 지분과 합산하여 경영권 지분율을 산정. [[지분구조]] 분석의 핵심 구성 요소.

## 유형

- 본인/배우자/직계 혈족
- 계열회사 (삼성물산, 삼성화재 등)
- 재단/기금
- 우리사주조합
- 임원진

## OPM에서의 처리

- own_major: 특수관계인 전체 리스트, relate 필드로 관계 구분. [[DART-OpenAPI]] DS002 hyslrSttus API 사용
- 지분율 1% 이상만 상세 표시
- 합계 행 별도 표시
