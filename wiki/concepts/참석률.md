# 참석률

## 개념

주주총회에서 [[의결권]]을 실제 행사한 주식의 비율. [[DART-OpenAPI]]에 직접 공시되지 않으며, [[KRX-KIND]] 투표결과에서 역산. [[KIND-주총결과]] 분석의 핵심 산출 지표.

## 역산 공식

```
전체 참석률 = 발행기준 찬성률 / 행사기준 찬성률
```

- 보통결의 안건 중 최빈값(mode)이 대표 참석률
- [[감사위원-의결권-제한]]으로 감사위원 선임은 분모 다름

## 최대주주 제외 참석률

```
최대주주 제외 참석률 = (전체참석 - 특관인지분) / (의결권주식 - 특관인지분)
```

## KOSPI 200 통계 (2026 정기주총)

| 항목 | 값 |
|------|-----|
| 평균 참석률 | 73.3% |
| 중위 참석률 | 75.1% |
| 최소 | 30.4% (호텔신라) |
| 최대 | 94.3% (삼성카드) |

## OPM에서의 처리

- agm_result: KIND 크롤링 -> 안건별 투표결과 + 추정참석률
- 정확한 행사 주식수는 DART 미공시 -> 추정치로 제공
