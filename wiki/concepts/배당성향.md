# 배당성향 (Payout Ratio)

## 개념

기업이 벌어들인 순이익 중 배당금으로 지급한 비율.

```
배당성향(%) = 배당금 총액 / 지배주주 귀속 당기순이익 x 100
```

## 핵심 규칙

- **반드시 지배주주 귀속 당기순이익 사용** (연결재무제표 기준)
- 비지배지분 순이익 포함 시 배당성향이 과소 계산됨
- 별도 재무제표의 당기순이익 사용 금지
- [[DART-OpenAPI]] alotMatter에 배당성향이 있으면 해당 값 우선 사용
- 당기순이익이 적자면: "적자 배당" 문자열 표시 (음수% 아님)

## 거버넌스 분석에서의 의미

배당성향은 주주환원 정책의 핵심 지표. 너무 낮으면 주주가치 훼손 논란, 너무 높으면 투자 여력 부족. 기관투자자의 의결권 행사 기준에 영향.
