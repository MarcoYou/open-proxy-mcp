# 배당수익률 (Dividend Yield)

## 개념

주가 대비 배당금의 비율. 투자자가 현재 주가로 매수했을 때 받을 수 있는 배당 수익.

## 두 가지 계산 방식

### DART 공식 시가배당률 (우선 사용)
```
시가배당률(%) = 1주당 배당금 / 기준주가 x 100
기준주가 = 배당기준일 전전거래일부터 과거 1주일간 종가의 산술평균
```
- [[시가배당률]] 참조

### 자체 계산 (DART 값 없을 때)
```
배당수익률(%) = 연간 DPS / 배당기준일 종가 x 100
```
- 종가: [[네이버-금융]] 시세 API (배당기준일 또는 직전 거래일)
- 1일 종가 vs 1주일 평균이므로 DART 값과 차이 가능

## 거버넌스 분석에서의 의미

배당수익률은 기업의 주주환원 매력도를 가격 대비로 평가하는 지표. 동일 DPS라도 주가가 하락하면 배당수익률은 상승하므로, 반드시 [[배당성향]]과 함께 해석해야 함. [[분기배당]] 기업은 연간 DPS 합산이 필요.

## OPM에서의 처리

- KRX 종가 없으면 DART 시가배당률 사용
- 둘 다 없으면 "-" 표시
