# 분기배당

## 개념

연 1회 결산배당 외에, 분기별로 중간배당을 실시하는 제도. [[배당성향]]과 [[배당수익률]] 계산 시 분기별 DPS 합산에 주의가 필요.

## 연간 DPS 계산

```
연간 DPS = 1Q DPS + 반기 DPS + 3Q DPS + 기말 DPS
```

- 각 분기 배당은 **해당 분기분만** (누적 아님)
- DART reprt_code로 구분: 11013(1Q), 11012(반기), 11014(3Q)

## 데이터 소스 주의

| 소스 | DPS 범위 |
|------|----------|
| DART alotMatter (사업보고서) | 연간 합산 |
| 현금배당결정 공시 | 해당 회차분만 |

예시 - 삼성전자 2025: 1Q 366 + 2Q 366 + 3Q 370 + 결산 566 = 1,668원

## OPM에서의 처리

- div_history: flow(개별 분기 DPS) + stock(연간 합산 DPS) 모두 표시. [[div-tool-rule]]에서 규칙 정의
- div_detail: [[DART-OpenAPI]] alotMatter 기준 연간 합산
- [[특별배당]]이 분기 중 포함될 수 있으므로 구분 주의
