# 5% 대량보유

## 개념

상장법인의 주식 5% 이상을 보유하게 된 자(보고자 + 특별관계자 합산)는 5일 이내에 금융위원회와 거래소에 보유상황 보고서를 제출해야 함 (자본시장법 제147조).

## 보유목적 분류

| 목적 | 의미 | 거버넌스 함의 |
|------|------|---------------|
| 단순투자 | 경영 관여 의사 없음 | 패시브 보유 |
| 일반투자 | 제한적 관여 가능 | 의결권 행사 가능 |
| 경영참여 | 이사 선임, 합병, 정관변경 등 | [[프록시-파이트]] 가능성 |

## OPM에서의 처리

- own_block: [[DART-OpenAPI]] majorstock API + 원문(document.xml) PUR_OWN 태그에서 보유목적 파싱
- DART majorstock API에 보유목적 필드가 없어 원문 파싱으로 해결
- "보고자 + 특별관계자 합산 기준" 명시
