# 국민연금

## 개요

한국 최대 기관투자자. 다수 KOSPI 200 기업에 [[5%-대량보유]] 이상 지분을 보유하며, [[프록시-파이트]]에서 핵심적인 [[의결권]] 행사 주체.

## OPM에서의 확인

- own_block: 국민연금의 대량보유 보고서 (stkrt + rcept_dt). [[DART-OpenAPI]] DS004 majorstock API 사용
- 보유목적: 통상 "단순투자" (경영참여가 아닌 패시브 보유)

## 주요 보유 사례

- 삼성전자: 11.52%
- KT&G: 7.50% (최대주주 IBK 8.06%보다 약간 적지만 실질적 영향력 큼)

## 거버넌스 분석에서의 의미

국민연금은 스튜어드십 코드를 통해 의결권 행사 방침을 공개. 주주가치 훼손 안건에 반대표를 던지는 경우가 증가 추세. 5% 이상 보유 기업의 주총에서 실질적 영향력 보유.
