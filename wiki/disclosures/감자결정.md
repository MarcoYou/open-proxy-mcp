# 감자결정

## 개요

이사회에서 자본금 감소(감자)를 결정할 때 제출하는 의무 공시. 재무구조 개선(결손보전), 합병 사전 조정, 주식병합 등 다양한 목적.

**감자 형태**:
- 무상감자: 대가 없이 주식수 감소 (결손 보전)
- 유상감자: 주주에게 대가 지급하며 주식 회수
- 주식병합: N주를 1주로 합치는 방식 (가장 흔함)

## 소스

- **제출처**: [[DART-OpenAPI]]
- **공시 유형**: pblntf_ty=B (주요사항보고) + I (거래소공시)
- **DART API**: `crDecsn.json` (DS005)
- **report_nm 패턴**: `주요사항보고서(감자결정)`, `감자결정`, `[기재정정]주요사항보고서(감자결정)`
- **의무 여부**: 의무
- **공시 주기**: 수시

## 전체 문서 구조 (트리)

```
주요사항보고서(감자결정)
│
├── 1. 감자할 주식
│   ├── 보통주 수                               ← crstk_ostk_cnt
│   ├── 종류주식 수                             ← crstk_estk_cnt
│   └── 1주당 액면가                            ← fv_ps
│
├── 2. 자본금 변화
│   ├── 감자 전 자본금                          ← bfcr_cpt
│   ├── 감자 후 자본금                          ← atcr_cpt
│   ├── 감자 전 발행주식수 (보통주)              ← bfcr_tisstk_ostk
│   ├── 감자 후 발행주식수 (보통주)              ← atcr_tisstk_ostk
│   └── 종류주식 동일                           ← bfcr_tisstk_estk / atcr_tisstk_estk
│
├── 3. 감자비율                                 ← cr_rt_ostk (보통주) / cr_rt_estk (종류주)
│
├── 4. 감자 기준일 & 방법
│   ├── 감자 기준일                             ← cr_std
│   └── 감자 방법 (주식병합 N:1, 액면가 감액 등)   ← cr_mth
│
├── 5. 감자 사유                                 ← cr_rs
│
├── 6. 일정
│   ├── 주주총회 예정일                         ← crsc_gmtsck_prd
│   ├── 명의개서정지 기간                       ← crsc_trnmsppd / cdobprpd_bgd/edd
│   ├── 구주권 제출기간                         ← crsc_osprpd_bgd / crsc_osprpd_edd
│   ├── 매매거래 정지기간                        ← crsc_trspprpd_bgd / crsc_trspprpd_edd
│   ├── 신주권 교부일                           ← crsc_nstkdlprd
│   └── 신주 상장 예정일                        ← crsc_nstklstprd
│
└── 7. 구주권 제출 대상 없는 경우 (대체)          ← ospr_nstkdl_pl
```

## API 필드 vs 원문 대응

| API 필드 | 원문 위치 | 의미 | 주의사항 |
|----------|-----------|------|----------|
| `cr_rt_ostk` | 3. 감자비율 | 보통주 감자비율 (%) | 83.33% = 6주→1주 병합 |
| `cr_rt_estk` | 3. 감자비율 | 종류주식 감자비율 | |
| `cr_std` | 4. 기준일 | 감자 기준일 | |
| `cr_mth` | 4. 방법 | 감자 방법 텍스트 | "액면가 100원 보통주 6주를 1주로 병합" |
| `cr_rs` | 5. 사유 | 감자 사유 | **"재무구조 개선" / "회생계획" → 재무위기 신호** |
| `bfcr_cpt` / `atcr_cpt` | 2. 자본금 | 전/후 자본금 (원) | 차액 = 자본금 감소액 |

| `crsc_trspprpd_bgd/edd` | 6. 매매정지 | 매매거래 정지기간 | **보통 2-3주** |

## OPM에서의 활용

- **`dilutive_issuance(scope="capital_reduction")`**: 카드 형태로 감자비율/자본금 변화/사유/일정 표시
- **`dilutive_issuance(scope="summary")`**: timeline에 "감자비율 X% / 사유" headline

## 거버넌스 분석 포인트

1. **감자 사유 (`cr_rs`)** 분류:
   - **재무구조 개선 / 결손 보전**: 적자 누적 → 자본잠식 회피
   - **회생계획 인가**: 법정 관리, 투자자에게 강한 부정 신호
   - **합병 사전 조정**: 합병비율 조정 목적
   - **잉여금 배당**: 유상감자 (사실상 현금 배당)
2. **감자비율 (`cr_rt_ostk`)**:
   - 10% 이하: 경미
   - 50%+: 심각 (기존주주 주식수 대폭 감소)
   - 83.33% (6:1) / 90% (10:1) 등: 회생 기업 패턴
3. **주식병합 N:1**:
   - 단주(1주 미만) 발생 처리: 시가로 강제 매각 → 소수주주 권리 희석
4. **자본금 vs 자본총계**: 감자는 자본금만 줄이고 자본잉여금으로 전환 가능 (손실 보전 없이 숫자상 정리)
5. **감자 후 유상증자 세트**: 먼저 감자로 자본잠식 해소 → 바로 신주 발행 → 3자배정 → 최대주주 변경 패턴 (EDGC 사례)
