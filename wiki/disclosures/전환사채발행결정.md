# 전환사채(CB) 발행결정

## 개요

이사회에서 전환사채(주식 전환권 부착 사채)를 발행할 때 제출하는 의무 공시. 표면금리·전환가·전환비율·refixing 조항이 핵심.

**잠재 희석 도구**: 전환권 행사 시 신주 발행 → 기존주주 희석. 사모 발행 + 낮은 전환가 + refixing(시가 하락 시 전환가 하향조정)이 결합되면 강한 희석 위험.

## 소스

- **제출처**: [[DART-OpenAPI]]
- **공시 유형**: pblntf_ty=B (주요사항보고)
- **DART API**: `cvbdIsDecsn.json` (DS005)
- **report_nm 패턴**: `주요사항보고서(전환사채권발행결정)`, `[기재정정]주요사항보고서(전환사채권발행결정)`
- **의무 여부**: 의무
- **공시 주기**: 수시

## 전체 문서 구조 (트리)

```
주요사항보고서(전환사채권발행결정)
│
├── 1. 사채 기본정보
│   ├── 회차                                    ← bd_tm
│   ├── 사채 종류 (사모/공모, 보증/무보증 등)    ← bd_knd
│   ├── 발행 총액                               ← bd_fta
│   ├── 발행방법 (사모 / 공모)                   ← bdis_mthn
│   └── 자본시장법 잔여 한도                     ← atcsc_rmislmt / rmislmt_lt70p
│
├── 2. 금리·만기
│   ├── 표면금리 (Coupon)                       ← bd_intr_ex
│   ├── 만기수익률 (YTM)                        ← bd_intr_sf
│   └── 만기일                                  ← bd_mtd
│
├── 3. 전환 조건
│   ├── 전환비율                                ← cv_rt
│   ├── 전환가격                                ← cv_prc
│   ├── 전환 대상 주식 종류                      ← cvisstk_knd
│   ├── 전환 시 발행될 주식수                    ← cvisstk_cnt
│   ├── 발행주식 총수 대비 비율 (잠재 희석률)     ← cvisstk_tisstk_vs ★
│   ├── 전환청구 가능기간 시작                   ← cvrqpd_bgd
│   ├── 전환청구 가능기간 종료                   ← cvrqpd_edd
│   ├── 시가하락 시 전환가 하한 (refixing 한도)  ← act_mktprcfl_cvprc_lwtrsprc
│   └── refixing 산정 근거                      ← act_mktprcfl_cvprc_lwtrsprc_bs
│
├── 4. 자금조달 목적                             ← fdpp_*
│
├── 5. 일정
│   ├── 사채 발행일 (납입일)                     ← sbd / pymd
│   └── 풋옵션 행사 가능일 (해당 시)
│
├── 6. 보증·담보
│   ├── 지급보증인                              ← rpmcmp
│   └── 담보                                    ← grint
│
├── 7. 해외발행 여부                             ← ovis_ltdtl / ovis_*
│
└── 8. 증권신고서 제출 여부                       ← rs_sm_atn / ex_sm_r
```

## API 필드 vs 원문 대응

| API 필드 | 원문 위치 | 의미 | 주의사항 |
|----------|-----------|------|----------|
| `bd_tm` | 1. 회차 | 회차 (예: 14회차) | |
| `bd_knd` | 1. 종류 | 사채 종류 텍스트 | "무기명식 무보증 사모 전환사채" 등 |
| `bd_fta` | 1. 총액 | 발행총액 (원) | |
| `bdis_mthn` | 1. 방식 | 사모 / 공모 | **사모는 우호 인수자 결정 가능** |
| `bd_intr_ex` | 2. 표면금리 | 액면 이자율 | 0% 흔함 (전환권 가치로 보전) |
| `bd_intr_sf` | 2. YTM | 만기수익률 | 풋옵션 행사 시 보장 수익률 |
| `bd_mtd` | 2. 만기 | 만기일 | |
| `cv_rt` | 3. 전환비율 | 보통 100% | |
| `cv_prc` | 3. 전환가 | 1주당 전환가격 (원) | **시가 대비 할인/할증 분석** |
| `cvisstk_cnt` | 3. 신주 | 전환 시 발행 신주수 | |
| `cvisstk_tisstk_vs` | 3. 비율 | 발행주식 총수 대비 비율 (%) | **잠재 희석률 핵심 지표** |
| `cvrqpd_bgd` | 3. 청구기간 | 전환청구 가능 시작 (보통 발행 후 1년) | |
| `act_mktprcfl_cvprc_lwtrsprc` | 3. refixing 한도 | 시가 하락 시 전환가 하한 | **하한이 낮으면 무한 희석 위험** |
| `rmislmt_lt70p` | 1. 잔여한도 | 자본시장법 한도 70% 이하 여부 | |

## OPM에서의 활용

- **`dilutive_issuance(scope="convertible_bond")`**: 카드 형태로 회차/총액/전환가/잠재 희석률/refixing/자금목적 모두 표시
- **`dilutive_issuance(scope="summary")`**: 다른 희석성 증권과 함께 timeline. headline에 "총액 / 전환가 / 잠재 희석 X%" 노출

## 거버넌스 분석 포인트

1. **잠재 희석률 (`cvisstk_tisstk_vs`)**:
   - 5% 이하: 경미
   - 10-30%: 주목
   - 30%+: 심각 (소수주주 희석 위험)
2. **사모 발행 (`bdis_mthn=사모`)**:
   - 발행 대상이 결정됨 → **우호 인수자 식별 필요** (원문 파싱 또는 공시 본문 확인)
   - 사모 발행 시 1년간 전환·권면분할 금지
3. **표면금리 0% + YTM 양수**: 전환권 가치를 인정. 전환가 적정성 평가 핵심.
4. **Refixing 조항 (`act_mktprcfl_cvprc_lwtrsprc`)**:
   - 한도 = 발행 시 전환가의 70% 미만이면 무한 희석 가능 (악성 CB)
   - 행동주의 펀드의 메자닌 자금 조달 패턴
5. **만기 vs 풋옵션**: 풋옵션이 짧으면 (1-2년) 단기 차익 추구. 길면 장기 자본조달.
6. **자금 목적**: 운영자금/채무상환 → 재무위기 신호. 사업양수/시설 → 성장.
