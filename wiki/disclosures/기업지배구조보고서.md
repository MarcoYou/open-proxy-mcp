# 기업지배구조보고서

## 개요

한국거래소가 제정한 15개 **지배구조 핵심지표**와 세부원칙 준수 여부를 종합 보고하는 공시. 거버넌스 종합 평가의 표준 문서.

## 의무 제출 대상 — 연도별 확대 일정

KRX가 자산 규모에 따라 단계적으로 확대해 2026년에 KOSPI 전체 의무화.

| 시기 | 대상 자산 규모 |
|------|---------------|
| 2019년 | 유가증권시장 자산 **2조원 이상** |
| 2020년 | 1조원 이상 |
| 2021년 | 5천억원 이상 (일부 시기) |
| 2022년 | **1조원 이상** (재조정) |
| 2024년 제출부터 | **자산 5천억원 이상** (2023 사업연도 기준, 가이드라인 개정) |
| **2026년 제출부터** | **KOSPI 전체 상장사** (2025 사업연도 기준) |

**KOSDAQ**: 현재 의무화 계획 확정 안 됨 → **자율공시**. 실제 제출 기업은 일부.

## 제출 시기 & 정정

- **정기 제출 시한**: 매년 **5월말** (직전 사업연도 기준)
  - 예: 2025 사업연도 보고서 → 2026년 5월 31일까지
- **연중 정정 빈번**: 오타·수치 오류·서식 개정 반영 등으로 `[기재정정]기업지배구조보고서공시` 형태로 재제출
  - 파싱 시 최신 rcept_dt 건을 우선
  - 정정 여부는 `report_nm.startswith("[기재정정]")`로 식별

## 의무화 시나리오 (OPM 분석 영향)

- **2025년 제출**까지 (2024 사업연도 기준): 자산 5천억 이상만 제출
  → 중소형 KOSPI 상장사는 데이터 **없음** 가능
- **2026년 제출부터** (2025 사업연도 기준): KOSPI 전체 제출 → 데이터 커버리지 확대
- KOSDAQ은 자율공시 — 일부만 데이터 있음 (주로 대형 KOSDAQ 상장사가 자발 제출)

## 소스

- **제출처**: [[DART-OpenAPI]] + [[KRX-KIND]] (병행)
- **공시 유형**: pblntf_ty=I (거래소공시), rcept_no `80` 포맷
- **DART API**: 전용 구조화 API 없음. `list.json` + 키워드 매칭 + 원문 파싱
- **report_nm 패턴**: `기업지배구조보고서공시`
- **제출 시기**: 보통 5~6월 (사업연도 종료 후)

## 15개 핵심지표 (2024 기준)

### 주주 부문 (5개)
1. 주주총회 4주 전에 소집공고 실시
2. 전자투표 실시
3. 주주총회의 집중일 이외 개최
4. 현금 배당관련 예측가능성 제공
5. 배당정책 및 배당실시 계획을 연 1회 이상 주주에게 통지

### 이사회 부문 (6개)
6. 최고경영자 승계정책 마련 및 운영
7. 위험관리 등 내부통제정책 마련 및 운영
8. 사외이사가 이사회 의장인지 여부
9. 집중투표제 채택
10. 기업가치 훼손 또는 주주권익 침해에 책임이 있는 자의 임원 선임 방지 정책
11. 이사회 구성원 모두 단일성(性)이 아님 (여성이사 확보)

### 감사기구 부문 (4개)
12. 독립적인 내부감사부서 설치
13. 내부감사기구에 회계 또는 재무 전문가 존재
14. 내부감사기구가 분기별 1회 이상 경영진 참석 없이 외부감사인과 회의 개최
15. 경영 관련 중요정보에 내부감사기구가 접근할 수 있는 절차 마련

## API 필드 vs 원문 대응

전용 API 없음. list.json 기본 필드:

| API 필드 | 의미 | 비고 |
|----------|------|------|
| `rcept_no` | 접수번호 | `80` 포맷 (거래소 수시공시) |
| `corp_code`, `corp_name` | 회사 | |
| `report_nm` | 제목 | `기업지배구조보고서공시` |
| `rcept_dt` | 접수일 | 보통 5-6월 |

**원문 파싱 필수 항목** (OPM `corp_gov_report`가 추출):
- 준수율 (%)
- 기업개요 (최대주주, 지분율, 소액주주 지분율, 기업집단, 요약재무)
- 15개 핵심지표 (지표명 + O/X + 직전기 O/X + 비고)
- 세부원칙별 응답 (100자 이내 텍스트)

## OPM에서의 활용

- **`corp_gov_report(scope="summary")`**: 기업개요 + 준수율 + 15지표 ✅/❌ 요약
- **`corp_gov_report(scope="metrics")`**: 15지표 당기·직전기 비교 전수
- **`corp_gov_report(scope="principles")`**: 세부원칙별 응답 텍스트
- **`corp_gov_report(scope="filings")`**: 연도별 제출 이력

## 거버넌스 분석 포인트

1. **준수율 추이**: 연도별 준수율 변화 (개선/후퇴 신호)
2. **구조적 미준수 항목**:
   - `집중투표제 채택 X`: 소수주주 보호 약함
   - `사외이사 이사회 의장 X`: 이사회 독립성 약함
   - `여성이사 0명`: 이사회 다양성 부족
   - `독립 감사부서 미설치`: 감사 독립성 약함
3. **동종 업계 비교**: 동일 섹터 기업의 준수율 기준선
4. **직전기 대비 변화**: 준수 → 미준수 전환은 거버넌스 후퇴
5. **100자 응답의 허술함**: "당사는 ~노력하고 있음" 같은 모호한 응답 vs 구체적 실행 증거
