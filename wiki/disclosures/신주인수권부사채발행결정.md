# 신주인수권부사채(BW) 발행결정

## 개요

이사회에서 신주인수권부사채(주식 인수권 부착 사채)를 발행할 때 제출하는 의무 공시. **사채 + 워런트** 구조: 만기까지 이자 수령 + 별도로 신주 인수권 행사 가능.

분리형 BW(워런트 분리 유통 가능) / 비분리형 BW로 구분. [[전환사채발행결정|CB]]와 달리 **사채 원금은 만기까지 유지**되고 워런트만 별도 행사.

## 소스

- **제출처**: [[DART-OpenAPI]]
- **공시 유형**: pblntf_ty=B (주요사항보고)
- **DART API**: `bdwtIsDecsn.json` (DS005)
- **report_nm 패턴**: `주요사항보고서(신주인수권부사채권발행결정)`
- **의무 여부**: 의무
- **공시 주기**: 수시

## API 필드 vs 원문 대응

| API 필드 | 원문 위치 | 의미 | 주의사항 |
|----------|-----------|------|----------|
| `bd_tm` | 1. 회차 | 회차 | |
| `bd_knd` | 1. 종류 | 사채 종류 | "무기명식 이권부 무보증 사모 신주인수권부사채" 등 |
| `bd_fta` | 1. 총액 | 발행총액 (원) | |
| `bdis_mthn` | 1. 방식 | 사모 / 공모 | **사모는 우호 인수자 결정** |
| `ex_rt` | 3. 행사비율 | 보통 100% | |
| `ex_prc` | 3. 행사가 | 주당 행사가 (원) | |
| `ex_prc_dmth` | 3. 행사가 결정방법 | "이사회 결의일 전일부터 1개월 가중평균" 등 | |
| `bdwt_div_atn` | 3. 분리/비분리 | 분리형 / 비분리형 | **분리형은 워런트 별도 유통, 경영권 방어 도구 가능** |
| `nstk_isstk_cnt` | 3. 신주 | 신주 수 | |
| `nstk_isstk_tisstk_vs` | 3. 비율 | 발행주식 총수 대비 (%) | **잠재 희석률 핵심 지표** |
| `nstk_pym_mth` | 3. 납입방법 | 대용납입 / 현금 | **대용납입은 사채로 신주대금 지급** |
| `expd_bgd` / `expd_edd` | 3. 행사기간 | 행사 가능 기간 | 보통 발행 후 1년부터 만기 전까지 |

## OPM에서의 활용

- **`dilutive_issuance(scope="warrant_bond")`**: 카드 형태로 회차/총액/행사가/잠재 희석률/분리형 여부/자금목적
- **`dilutive_issuance(scope="summary")`**: timeline에 "총액 / 행사가 / 희석 X% / 분리형" headline

## 거버넌스 분석 포인트

1. **분리형 BW (`bdwt_div_atn=분리`)** ★:
   - 워런트가 사채와 분리되어 별도 유통
   - 사채만 금리 수익, 워런트는 별도로 주식 전환 권리 매매
   - 경영권 방어 목적 우호 세력에 워런트 분배 사례 다수
2. **비분리형 BW**:
   - 사채와 워런트가 같이 움직임 → 순수 자금조달
3. **대용납입 (`nstk_pym_mth=대용납입`)**:
   - 워런트 행사 시 신주대금을 **사채 원금으로 납입** (사채 소멸)
   - 실질적으로 CB와 유사한 효과
4. **잠재 희석률 (`nstk_isstk_tisstk_vs`)**:
   - CB와 동일 기준
   - 5% 이하 경미 / 10-30% 주목 / 30%+ 심각
5. **행사가 결정방법**:
   - "1개월 가중평균 최저가" → 낮은 행사가 확보 (워런트 보유자 유리)
   - "이사회결의일 종가" → 공정
6. **표면금리 + YTM**: CB와 같지만 BW는 사채와 워런트가 분리돼 있어 가격 구조 더 복잡
