# 유상증자결정

## 개요

이사회에서 신주를 발행해 자본 조달을 결정할 때 제출하는 의무 공시. 배정방식(주주배정/제3자배정/일반공모)과 자금 사용목적이 거버넌스 핵심.

**제3자배정 유상증자**는 우호 지분 형성·기존주주 희석 도구로 자주 사용됨.

## 소스

- **제출처**: [[DART-OpenAPI]]
- **공시 유형**: pblntf_ty=B (주요사항보고)
- **DART API**: `piicDecsn.json` (DS005)
- **report_nm 패턴**: `주요사항보고서(유상증자결정)`, `[기재정정]주요사항보고서(유상증자결정)`
- **의무 여부**: 의무
- **공시 주기**: 수시

## 전체 문서 구조 (트리)

```
주요사항보고서(유상증자결정)
│
├── 1. 신주의 종류와 수
│   ├── 보통주 신주 수                          ← nstk_ostk_cnt
│   ├── 종류주식 신주 수                        ← nstk_estk_cnt
│   └── 1주당 액면가                            ← fv_ps
│
├── 2. 증자 전 발행주식 (희석률 계산용)
│   ├── 보통주 총수                             ← bfic_tisstk_ostk
│   └── 종류주식 총수                           ← bfic_tisstk_estk
│
├── 3. 자금조달 목적                             ← fdpp_*
│   ├── 시설자금 (fdpp_fclt)
│   ├── 사업양수자금 (fdpp_bsninh)
│   ├── 운영자금 (fdpp_op)
│   ├── 채무상환자금 (fdpp_dtrp)
│   ├── 타법인 증권 취득자금 (fdpp_ocsa)
│   └── 기타 (fdpp_etc)
│
├── 4. 증자방식                                 ← ic_mthn
│   └── 주주배정 / 제3자배정 / 일반공모 / 주주우선공모
│
├── 5. 보호예수 (제3자배정 시)
│   ├── 보호예수 여부                           ← ssl_at
│   ├── 보호예수 시작일                         ← ssl_bgd
│   └── 보호예수 종료일                         ← ssl_edd
│
└── 추가 항목 (원문 본문):
    - 발행가, 기준주가
    - 신주배정 기준일
    - 청약일, 납입일
    - 신주 상장 예정일
    - 제3자배정 시 배정 대상자 명세
```

## API 필드 vs 원문 대응

| API 필드 | 원문 위치 | 의미 | 주의사항 |
|----------|-----------|------|----------|
| `nstk_ostk_cnt` | 1. 신주 | 신주 발행 보통주 수 | |
| `nstk_estk_cnt` | 1. 신주 | 신주 발행 종류주식 수 | |
| `fv_ps` | 1. 액면가 | 1주당 액면가 (원) | |
| `bfic_tisstk_ostk` | 2. 증자 전 | 기존 보통주 발행수 | **희석률 = 신주 / 기존** |
| `ic_mthn` | 4. 방식 | 증자방식 | 제3자배정이 거버넌스 신호 |
| `fdpp_op` | 3. 자금목적 | 운영자금 | 가장 흔한 목적 |
| `fdpp_ocsa` | 3. 자금목적 | 타법인 증권 취득 자금 | M&A 자금조달 신호 |
| `fdpp_dtrp` | 3. 자금목적 | 채무상환 | 재무위기 신호 가능 |
| `ssl_at` | 5. 보호예수 | 보호예수 약정 (Y/N) | 제3자배정 시 핵심 |

**원문 파싱 필요 (API 미제공)**:
- 발행가, 기준주가, 시가 대비 할인율
- 제3자배정 대상자 (개인/법인 명세)
- 신주 상장 예정일

## OPM에서의 활용

- **`dilutive_issuance(scope="rights_offering")`**: 카드 형태로 배정방식 / 신주 수 / 희석률 / 자금목적 / 보호예수 노출
- **`dilutive_issuance(scope="summary")`**: CB/BW/감자와 함께 timeline 통합. headline_metric에 "방식 / 신주 N주 / 희석률 X%" 노출
- **`screen_events`**: (향후) `rights_offering_decision` event_type 추가 가능
- **희석률 근사**: `dilution_pct_approx = nstk_ostk_cnt / bfic_tisstk_ostk × 100`

## 거버넌스 분석 포인트

1. **증자 방식 (`ic_mthn`)**:
   - **주주배정**: 모든 주주에게 비례 배정 → 희석 없음
   - **제3자배정**: 특정 자에게만 배정 → **기존주주 희석 + 우호지분 형성 가능**
   - 일반공모: 시장 공모, 중립
   - 주주우선공모: 주주 우선청약 후 잔여분 공모
2. **희석률**:
   - 5% 이하: 경미
   - 5-20%: 주목
   - 20%+: 심각 (EDGC 사례 272% — 회생절차 중)
3. **자금 목적의 우선순위**:
   - 채무상환 + 운영자금 → 재무위기 신호
   - 시설/사업양수 → 성장 투자
   - 타법인 주식 취득 → M&A
4. **보호예수**: 제3자배정 시 보호예수 여부와 기간이 우호 지분의 결속력 지표
5. **발행가 할인율** (원문 파싱): 시가 대비 큰 할인은 기존주주 가치 훼손
