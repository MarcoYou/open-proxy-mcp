# 배당 공시 유형

DART 거래소공시(`pblntf_ty="I"`)에서 배당 관련 주요 공시 유형 6종.
원문 파싱 기준: 2024-2026년 실제 제출 공시 트리 직접 확인.

---

## 1. 현금ㆍ현물배당결정

**설명**: 이사회에서 현금(또는 현물) 배당을 결정하면 제출하는 공시. 배당이벤트의 핵심 정보가 담긴 공시.

**샘플 rcept_no**: 20260129800004 (삼성전자, 2026-01-29), 20260129800384 (현대차), 20250513800515 (POSCO홀딩스 분기)

**문서 구조**:
```
[회사명]/현금ㆍ현물 배당 결정/([날짜])현금ㆍ현물 배당 결정

현금ㆍ현물배당 결정

1. 배당구분          결산배당 | 분기배당 | 중간배당
2. 배당종류          현금배당 | 현물배당
   - 현물자산의 상세내역    (현물배당 시만 기재)
3. 1주당 배당금(원)
   보통주식           XXX
   종류주식           XXX
   - 차등배당 여부      미해당 | 해당
4. 시가배당율(%)
   보통주식           X.X
   종류주식           X.X
5. 배당금총액(원)      X,XXX,XXX,XXX
6. 배당기준일         YYYY-MM-DD
7. 배당금지급 예정일자  YYYY-MM-DD | -
8. 주주총회 개최여부   개최 | 미개최
9. 주주총회 예정일자   YYYY-MM-DD | -
10. 이사회결의일(결정일) YYYY-MM-DD
    - 사외이사 참석여부    참석(명) N / 불참(명) N
    - 감사(사외이사가 아닌 감사위원) 참석여부  -
11. 기타 투자판단과 관련한 중요사항  (자유기재)

【종류주식에 대한 배당 관련 사항】  (우선주 있을 경우 추가 테이블)

```

**주요 필드**:
- `1. 배당구분`: 결산배당 / 분기배당 / 중간배당 / 특별배당
- `2. 배당종류`: 현금배당 / 현물배당
- `3. 1주당 배당금` (보통주/종류주식): DPS 핵심 필드
- `4. 시가배당율(%)`: 이사회결의일 직전 1주일 평균주가 기준
- `5. 배당금총액`: 전체 지급액 (원)
- `6. 배당기준일`: 권리주주 확정일
- `7. 배당금지급 예정일자`: 미정 시 `-`
- `8/9. 주주총회`: 결산배당은 주총 승인 필요, 분기배당은 `미개최`
- 우선주 별도 테이블: 하단 `【종류주식에 대한 배당 관련 사항】`

**OPM 활용**: `div_detail` 파싱 — 배당구분/DPS/기준일/지급일 추출. 분기배당 기업은 연간 4건.

## 2. 현금ㆍ현물배당을위한주주명부폐쇄(기준일)결정

**설명**: 배당 권리주주를 확정하는 기준일을 이사회에서 결정할 때 제출. 배당결정 공시와 동일일자 혹은 별도로 제출.

**샘플 rcept_no**: 20260130800518 (KB금융, 2026-01-30), 20260224801139 (미래에셋증권), 20250513800482 (POSCO홀딩스 분기)

**문서 구조**:
```
[회사명]/현금ㆍ현물배당을 위한 주주명부폐쇄(기준일) 결정/([날짜])...

현금ㆍ현물배당을 위한 주주명부폐쇄(기준일) 결정

1. 배당구분          결산 | 중간(분기)
2. 주주명부폐쇄(기준일)
   시작일            -  (폐쇄 없이 기준일만 쓰는 경우 -)
   종료일            -
   기준일            YYYY-MM-DD
3. 주주명부폐쇄(기준일) 목적    권리주주 확정
4. 이사회결의일(결정일) YYYY-MM-DD
   - 사외이사 참석여부    참석(명) N / 불참(명) N
   - 감사(사외이사가 아닌 감사위원) 참석여부  -
5. 기타 투자판단과 관련한 중요사항  (관련 정관 조문 등 자유기재)

※ 관련공시  (연결된 공시 — 주로 배당기준일 안내 선행공시 참조)
```

**주요 필드**:
- `1. 배당구분`: 결산 / 중간(분기) — 분기배당 기업은 분기마다 제출
- `2. 기준일`: 배당기준일 (실제 권리주주 확정일)
- 실무상 주주명부 폐쇄 없이 기준일만 설정하는 경우가 대부분 (`시작일/종료일: -`)

**OPM 활용**: 배당기준일 확정 정보. `현금ㆍ현물배당결정` 공시의 `6. 배당기준일`과 연동.

## 3. 주주명부폐쇄기간또는기준일설정

**설명**: 배당과 무관하게 정기주주총회 의결권 주주 확정 또는 분배금 지급을 위해 기준일을 설정하는 공시. 투자회사(펀드, 인프라 등)에서 분배금 목적으로도 사용.

**샘플 rcept_no**: 20251211800753 (맥쿼리인프라, 2025-12-11), 20251216801146 (두산, 2025-12-16), 20251216801810 (삼성물산, 2025-12-16)

**문서 구조**:
```
[회사명]/주주명부폐쇄기간 또는 기준일 설정/([날짜])...

주주명부폐쇄기간 또는 기준일 설정

1. 기준일            YYYY-MM-DD
2. 명의개서정지기간
   시작일            - (대부분 폐쇄 없음)
   종료일            -
3. 설정사유          (자유기재)
                     예) "제89기 정기주주총회 권리주주 확정"
                     예) "2025년 하반기 분배금 지급을 위한 권리주주 확정"
4. 기타 투자판단에 참고할 사항   (자유기재)

※ 관련 공시
```

**주요 필드**:
- `1. 기준일`: 권리주주 확정 기준일
- `3. 설정사유`: 주총 소집 목적인지 배당/분배금 목적인지 구분 가능
- 배당 목적 기준일설정과 주총 목적 기준일설정이 동일 서식 사용 — `설정사유` 텍스트로 구분

**주의**: 이 공시가 먼저 나오고 이후 `현금ㆍ현물배당결정` 공시가 나오는 경우가 많음 (특히 12월에 기준일 사전 공고, 1-2월에 배당금액 결정).

**OPM 활용**: 주총 기준일과 배당 기준일 혼재 — `설정사유` 파싱으로 구분 필수.

- 주총 안건으로 상정되어 주총 결과에 따라 변경 가능

**OPM 활용**: 희석 EPS 계산 시 주식배당 신주 수 반영 필요.

## 공시 유형 간 관계

```
기준일 사전 안내 (12월)
  └─ 주주명부폐쇄기간또는기준일설정 (주총용 or 배당용 혼재)
       └─ 현금ㆍ현물배당을위한주주명부폐쇄(기준일)결정 (배당기준일 확정)
            └─ 현금ㆍ현물배당결정 (배당금액 확정, 이사회결의일에 동시 또는 별도 제출)

분기배당 기업 (POSCO홀딩스, 현대차, 기아 등)
  └─ 분기마다: 현금ㆍ현물배당을위한주주명부폐쇄(기준일)결정 + 현금ㆍ현물배당결정
               (연 4회 반복)

금융지주 자회사
  └─ (자회사의 주요경영사항) 버전으로 별도 제출
     ─ 모회사 공시와 중복 집계 주의
```

## 파싱 주의사항

1. **배당구분 값 표준화**: `결산배당` / `분기배당` / `중간(분기)` / `결산` — 서식에 따라 표기가 다름. `중간(분기)`와 `분기배당`은 동일 의미.

2. **시가배당율(%)**: 이사회결의일 직전 1주일 평균주가 기준 (배당기준일 기준 아님). 분기배당은 이사회결의일 기준, 결산배당은 이사회 or 배당부 종가일 기준 — 기업마다 다름.

3. **주식배당결정의 1주당 배당주식수**: 소수점 7자리까지 표기. `발행주식총수`는 자기주식 포함 총수이므로 실제 배당 대상 주식수와 다름.

4. **주주명부폐쇄기간또는기준일설정의 이중 목적**: 주총 의결권 기준일 vs. 배당 기준일 — `3. 설정사유` 텍스트로 구분. 주총용이면 배당 기준일이 별도 공시로 나중에 나옴.

5. **[기재정정] 공시**: 원본 공시 제출 후 내용 정정 시 `[기재정정]` prefix 붙음. 정정 공시 내에 `정정 전/후` 표가 있고 하단에 정정된 전체 공시 내용이 다시 포함됨. 파싱 시 최신 `[기재정정]` 공시를 우선으로 사용.

6. **자회사판 중복 제거**: `(자회사의 주요경영사항)` 공시는 모회사 시각에서의 자회사 정보. 유니버스(KOSPI200)가 모회사 기준이면 이 공시의 배당금은 모회사에 귀속되는 배당 수입이지, 시장 배당 이벤트가 아님 — OPM div_detail에서 제외.
