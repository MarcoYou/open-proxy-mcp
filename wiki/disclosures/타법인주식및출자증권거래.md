# 타법인주식 및 출자증권 거래

## 개요

회사가 다른 법인의 주식 또는 출자증권을 양수(취득)하거나 양도(처분)할 때 제출하는 의무 공시. **자회사·관계회사에 대한 출자 변경** 및 그룹 내 재편의 주요 신호.

특히 **특수관계인 대상 거래**는 일감몰아주기·가치 이전의 핵심 지표.

## 공시 제목 4종

| 제목 | pblntf_ty | 구분 |
|------|-----------|------|
| 타법인주식및출자증권양수결정 | B | 주요사항보고서 |
| 타법인주식및출자증권양도결정 | B | 주요사항보고서 |
| 타법인주식및출자증권취득결정 | I | 거래소공시 (자율/자회사 포함) |
| 타법인주식및출자증권처분결정 | I | 거래소공시 |

변형:
- `[기재정정]...` 정정공시
- `...(자율공시)` 자율 공시
- `...(자회사의 주요경영사항)` 모회사 명의로 자회사 거래 공시

## 소스

- **제출처**: [[DART-OpenAPI]]
- **공시 유형**: pblntf_ty=B (주요사항보고) + I (거래소공시)
- **DART API**: 전용 구조화 API 없음. `list.json` + 제목 키워드 매칭
- **의무 여부**: 의무 (일정 금액·비율 초과 시)
- **공시 주기**: 수시

## 전체 문서 구조 (트리) — 원문

API에서 구조화 필드 없음. 원문 본문에서 아래 정보 파싱 필요:

```
타법인주식 및 출자증권 양수/양도/취득/처분 결정
│
├── 1. 거래 기본정보
│   ├── 거래 종류 (양수/양도/취득/처분)
│   ├── 거래 상대방 회사 (회사명, 대표자)
│   ├── 본 회사와의 관계 ★ (자회사/관계회사/특수관계인/제3자)
│   └── 거래 대상 주식 종류 (보통주/우선주/출자증권)
│
├── 2. 거래 규모
│   ├── 거래 주식수
│   ├── 주당 거래가격
│   ├── 총 거래금액 ★
│   ├── 자산총계 대비 비율 (%)
│   └── 자본총계 대비 비율 (%)
│
├── 3. 거래 후 지분 변화
│   ├── 거래 전 보유 주식수 및 비율
│   └── 거래 후 보유 주식수 및 비율
│
├── 4. 거래 상대방 재무 (해당 시)
│   └── 자산/부채/자본/매출/순이익
│
├── 5. 거래 목적 / 사유
│
├── 6. 자금조달 방법 (취득·양수의 경우)
│
└── 7. 일정
    ├── 이사회 결의일
    ├── 계약 체결일
    └── 대금 지급일
```

## API 필드 vs 원문 대응

list.json 기본 필드만 제공. 상세는 원문 파싱 필요.

| API 필드 | 의미 | 비고 |
|----------|------|------|
| `rcept_no`, `rcept_dt` | 접수번호/일 | |
| `corp_code`, `corp_name` | 발행회사 | |
| `report_nm` | 공시 제목 | 키워드 매칭 후 양수/양도 분류 |
| `flr_nm` | 제출인 | 발행회사 = 자회사 거래 공시 시 모회사명 |

**원문 파싱 필수 항목**:
- 거래 상대방 회사명
- 특수관계 여부
- 거래금액
- 자산/자본 대비 비율
- 지분 변화

## OPM에서의 활용

- **`related_party_transaction(scope="equity_deal")`**: list.json 메타만 수집. 방향(취득/처분), 자회사/자율공시 플래그 표시
- **`related_party_transaction(scope="summary")`**: 단일공급계약과 함께 timeline 통합
- **원문 상세**: `evidence(rcept_no=...)` 또는 AI가 DART 뷰어 링크 따라가서 직접 확인

## 거버넌스 분석 포인트

1. **자회사 주요경영사항 공시 (`(자회사의 주요경영사항)`)**:
   - 모회사가 비상장 자회사 거래를 대신 공시
   - POSCO홀딩스 같은 지주회사에서 빈번
   - 자회사 거버넌스 투명성 확보 수단이자, 내부거래 드러나는 지점
2. **자율공시 (`(자율공시)`)**:
   - 의무 공시 기준 미달이지만 자발적 공시
   - 투명성 의지 긍정 신호
3. **거래 상대방 특수관계 (원문 파싱)**:
   - **계열사 간 주식 이동** → 지배구조 재편 / 일감몰아주기 / 가치 이전
   - **대주주 개인 거래** → 사익 편취 위험
   - **제3자 거래** → 일반 투자 결정
4. **양수·취득 패턴** (acquire):
   - 자회사 지분 확대 → 100% 자회사 편입 준비
   - 관계회사 투자 → 사업 영역 확장
   - 특수관계인 주식 인수 → 우호 지분 / 대주주 이익
5. **양도·처분 패턴** (dispose):
   - 자회사 매각 → 사업 정리 / 구조조정
   - 주식 맞교환 → [[주식교환·이전결정]]와 연계
   - 현금화 → 유동성 확보
6. **[기재정정]** 빈번 여부: 정정 공시가 많으면 초기 공시 품질 의심
