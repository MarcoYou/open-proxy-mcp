# 최대주주등소유주식변동신고서

## 개요

| 항목 | 내용 |
|------|------|
| 제출처 | KRX 거래소 (KIND) |
| 공시 유형 | 수시공시 (pblntf_ty=I) |
| 제출 의무 | 최대주주+특수관계인 합산 지분 변동 시 |
| DART rcept_no | `80` 포맷 (거래소 수시공시) |
| KIND acptno | rcept_no 8번째 이후 `80` → `00` 변환 |

## 접근 경로

```
DART search (pblntf_ty=I, report_nm=최대주주등소유주식변동신고서)
  → rcept_no (80 포맷)
  → KIND acptno 변환 (80 → 00)
  → kind_fetch_document(acptno)
  → HTML 파싱
```

## 원문 구조 (5개 섹션)

### 1. 발행회사 정보
- 회사명, 회사코드(ticker), 담당부서, 담당자

### 2. 발행주식수 정보
- 보통주식총수 / 종류주식총수 / 발행주식총수

### 3. 보고의 개요
- 직전보고서 제출일 + 직전 소유주식 (주식수 + 비율)
- 금번보고서 제출일 + 금번 소유주식 (주식수 + 비율)
- → **순변동 계산 가능** (금번 - 직전)

### 4. 개인별 세부변동사항
- 주주별 테이블 (주주 수만큼 테이블 반복)
- 컬럼: 변경일 / 변경원인 / 주식의 종류 / 변경전주식수 / 증감주식수 / 변경후주식수 / 비고
- 변경원인 예: 장내매수, 장내매도, 장외매도, 무상증자, 상속 등

### 5. 최대주주등 주식소유현황 (총괄)
- 전체 최대주주+특관인 명단
- 컬럼: 성명 / 관계 / 보통주(주식수+비율) / 종류주(주식수+비율) / 합계(주식수+비율)

## 파싱 전략

```
BeautifulSoup lxml
  → find_all('span', style=lambda s: '11pt' in s and 'bold' in s)  # 섹션 헤더
  → find_all('table')  # 테이블 순서로 섹션 매핑
```

- 테이블 1 → 섹션 1 (발행회사)
- 테이블 2 → 섹션 2 (발행주식수)
- 테이블 3 → 섹션 3 (보고개요)
- 테이블 4~N-1 → 섹션 4 (개인별 변동, 주주 수만큼)
- 테이블 N → 섹션 5 (총괄현황)

## 주의사항

- 섹션 4 테이블 수 = 변동 있는 주주 수 (가변)
- 총괄현황(섹션 5)은 항상 마지막 테이블
- KIND 크롤링 간격 준수 (2-5초 랜덤)
- 화이트리스트 공시: KIND 접근 허용 대상에 추가 필요
