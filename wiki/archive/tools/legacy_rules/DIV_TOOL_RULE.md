# DIV Tool Rule

## Tool 구조 (5 tools)

### 오케스트레이터
- `div(ticker)` — 배당 종합 (최신 상세 + 3년 추이)

### 개별 Tool
- `div_search(ticker)` — 배당 관련 공시 검색 (현금배당 결정, 중간배당)
- `div_detail(ticker, bsns_year, reprt_code)` — 배당 상세 (DPS, 총액, 성향, 수익률)
- `div_history(ticker, years)` — 배당 이력 (연도별 추이)
- `div_manual()` — 이 문서 + DIV_CASE_RULE.md 반환

## 핵심 연산 규칙

### 배당성향 (Payout Ratio)
```
배당성향(%) = 배당금 총액 / 지배주주 귀속 당기순이익 × 100
```
- **반드시 '지배주주 귀속 당기순이익' 사용** (연결재무제표 기준)
- 비지배지분 순이익을 포함하면 배당성향이 과소 계산됨
- 별도 재무제표의 당기순이익 사용 금지 (연결 vs 별도 괴리 큼)
- DART `alotMatter`에 배당성향이 있으면 그 값을 우선 사용

### 배당수익률 (Dividend Yield)

**DART 공식 시가배당률 (우선 사용):**
```
시가배당률(%) = 1주당 배당금 / 기준주가 × 100
기준주가 = 배당기준일 전전거래일(배당부 종가일)부터 과거 1주일간 종가의 산술평균
```
- DART `alotMatter`와 현금배당결정 공시에 시가배당률이 제공됨
- 이 값이 있으면 그대로 사용

**자체 계산 (DART 값 없을 때):**
```
배당수익률(%) = 연간 DPS / 배당기준일 종가 × 100
```
- 종가: 네이버 금융 시세 API (배당기준일 또는 직전 거래일)
- 단순 종가 기반이므로 DART 공식 시가배당률과 다를 수 있음 (1주일 평균 vs 1일 종가)

### 배당 날짜 구분

| 날짜 | 의미 | 소스 |
|------|------|------|
| 결산일 (stlm_dt) | 사업연도 종료일 | DART alotMatter |
| 배당기준일 (record date) | 배당 수령 주주 확정일 | 현금배당결정 공시 본문 |
| 이사회결의일 | 배당 결정일 | 현금배당결정 공시 본문 |
| 배당금지급 예정일 | 실제 지급일 | 현금배당결정 공시 본문 (주총 후 1개월 이내) |

- 결산배당: 배당기준일 = 결산일 (통상 12/31)
- 중간배당: 배당기준일 ≠ 결산일 (이사회가 별도 정함)
- 분기배당: 분기 말일 (3/31, 6/30, 9/30)
- 배당 날짜 상세는 `div_search` → 현금배당결정 공시 본문에서 확인

### DPS 데이터 소스 주의

| 소스 | DPS | 범위 |
|------|-----|------|
| DART `alotMatter` (사업보고서) | **연간 합산** | 분기배당 포함 전체 |
| 현금배당결정 공시 (주요사항보고서) | **해당 회차분만** | 결산배당/분기배당 개별 |

예시 (삼성전자 2025):
- alotMatter DPS: 1,668원 (연간 합산)
- 현금배당결정 (결산): 566원 (4Q분만)
- 분기배당: 1Q 366 + 2Q 366 + 3Q 370 + 결산 566 = 1,668원

### 분기/반기 배당
```
연간 DPS = 1Q DPS + 반기 DPS + 3Q DPS + 기말 DPS
```
- 각 분기 배당은 해당 분기분만 (누적이 아님)
- flow(개별 분기 DPS) + stock(연간 합산 DPS) 모두 표시
- 분기배당 여부: reprt_code 11013(1Q), 11012(반기), 11014(3Q)에 데이터 있으면 해당

### 특별배당
- 정기배당과 별도. "특별" 키워드로 감지
- 연간 DPS 합산에 포함하되, 별도 행으로 표시
- 배당성향 계산 시 포함 (총배당금에 특별배당 포함)

### 우선주 (종류주식)
공시의 【종류주식】 섹션에서 복수 우선주를 배열로 파싱. `preferred_stocks` 필드.

**우선주 네이밍 규칙 (시장 관행 → 공시 표기):**

| 시장 호칭 | 공시 표기 | 분류 | 예시 |
|-----------|----------|------|------|
| {회사}우 | {회사}우 또는 우선주 | 구형(1우) | 삼성전자우, 현대차우 |
| {회사}2우B | 2우선주 또는 {회사}2우B | 신형(2우) | 현대차2우B → "2우선주" |
| {회사}3우B | 3우선주 또는 {회사}3우B | 신형(3우) | 한화3우B → "한화3우B" |
| 1종우선주 | 제1종우선주식 또는 1우선주 | 구형 | CJ제일제당, 녹십자홀딩스 |
| 전환우선주 | {회사}N우(전환) 또는 제N차 | 전환 | CJ4우(전환), 키움증권 제3차 |

유저가 "2우B"라고 물으면 → `preferred_stocks`에서 "2우선주" 또는 "2우B" 포함 항목을 매칭.
유저가 "우선주"라고만 하면 → 구형(1우) 기준으로 답변.

**복수 우선주 기업 (KOSPI 200):**
- 녹십자홀딩스: 1우선주 + 2우선주
- 두산: 두산우 + 두산2우B
- 한화: 한화우 + 한화3우B
- 키움증권: 제3차 + 제4차 (전환)
- 현대차: 우선주 + 2우선주 + 3우선주

### 주식배당
- 데이터 관리하되 현금배당 대비 중요도 낮음
- 별도 행으로 표시, DPS 합산에는 미포함

## 데이터 소스

| 데이터 | 소스 | API/필드 |
|--------|------|----------|
| 주당배당금(DPS, 연간) | DART | `alotMatter` → 주당 현금배당금(원) |
| 주당배당금(DPS, 회차별) | DART | 현금배당결정 공시 → 1주당 배당금 |
| 배당금 총액 | DART | `alotMatter` → 현금배당금총액(백만원) |
| 배당성향 (연결) | DART (우선) | `alotMatter` → (연결)현금배당성향(%) |
| 배당성향 (계산) | DART | 배당금총액 / 지배주주귀속 당기순이익 |
| 시가배당률 | DART (우선) | `alotMatter` → 현금배당수익률(%) — 기준일 전전거래일 1주 평균 |
| 배당수익률 (계산) | KRX | DPS / KRX 종가 × 100 |
| 종가 | KRX | `stk_bydd_trd` → `TDD_CLSPRC` |
| 지배주주 당기순이익 | DART | 사업보고서 연결 IS 또는 `agm_financials_xml` |

## API 호출

| Tool | DART API | KRX API |
|------|----------|---------|
| div_search | search_filings × 1 | - |
| div_detail | alotMatter × 1 | - |
| div_history | alotMatter × (years × 4) | stk_bydd_trd × years |
| div (종합) | div_detail + div_history | div_history 포함 |

주의: div_history는 연도당 최대 4회 DART API 호출 (기말+3분기). 3년 = 최대 12회.
