## 개요

임원 및 주요주주(10% 이상)의 특정증권 소유 변동을 신고하는 의무 공시. 내부자 거래 감시의 핵심 자료.

## 소스

- **제출처**: [[DART-OpenAPI]] (DS004 elestock API)
- **의무 여부**: 의무 (변동 발생 시 5일 이내)
- **공시 주기**: 수시 (소유 변동 시)

## 주요 내용

- 보고자(임원/주요주주) 정보
- 변동 전/후 소유 현황
- 변동 사유 (장내매매, 스톡옵션 행사 등)
- 특정증권 종류 (보통주, 전환사채 등)

## 전체 문서 구조 (트리)

```
임원·주요주주 특정증권 소유상황보고서
├── 보고자 정보 (임원/주요주주 구분)
├── 보유 증권 현황
│   ├── 주권 (보통주/우선주)
│   ├── 스톡옵션 (주식매수선택권)
│   └── 기타 특정증권
├── 변동 내역
│   ├── 취득/처분 일자 및 방법
│   └── 단가
└── 보유 목적
```

API: DART DS004 elestock
주의: 임원 내부자 거래 추적용. 대량보유(5%)와 다름 — 임원이면 1주 변동도 보고 의무

## API 필드 vs 원문 대응

| API 필드 | 원문 위치 | 실제 의미 | 주의사항 |
|----------|-----------|-----------|----------|
| `repror` | 표지 > 보고자 | 보고자 이름 | 임원/주요주주 구분 필요 |
| `stkcd` | 보유 증권 현황 | 종목코드 | |
| `stkqy` | 변동 내역 | 보유 주식 수 | 변동 후 기준 |
| `change_stkqy` | 변동 내역 | 변동 주식 수 | 취득 양수, 처분 음수 |
| `change_reason` | 변동 내역 | 변동 사유 | 장내매매/스톡옵션 행사 등 |

**대량보유상황보고서(5% 이상)와 별개 공시** — 임원은 보유 비율 무관하게 1주 변동도 신고 의무.

## 파싱 주의사항

- 임원 vs 주요주주(10% 이상) 구분하여 분석 목적에 따라 필터링
- 스톡옵션 행사는 주식 수 증가로 표시되나 시장 매수와 성격이 다름 — 변동 사유 반드시 확인
- 특수관계인의 보유분이 합산되지 않으므로, 실질 지배력 분석 시 [[대량보유상황보고서]]와 병행 사용

## OPM에서의 활용

- own_latest: [[최대주주]]와 [[특수관계인]]의 최근 지분 변동 참조
- [[own-tool-rule]]에서 수시 공시 소스로 활용
- 내부자의 매도/매수 패턴으로 경영권 변동 신호 감지 가능
