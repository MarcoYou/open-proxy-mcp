# 정관 우회 시나리오 catalog (2026 상법 개정 대응)

## 배경

2026 상법 개정 (1·2·3차)으로 자산 2조+ 상장사 중심 강행규정 적용. 기업 측은 법 테두리 안에서 우회 전술을 시도할 수 있음. OPM은 이를 4 시나리오로 catalog하고 36 항목으로 분류.

## 시나리오 1: 집중투표/분리선출 무력화

### 법 정합 의무
- 자산 2조+: 집중투표 의무화 (정관 배제 불가) + 분리선출 2명 이상

### 우회 전술 (방어자 입장)

**1-A) 정관변경 통한 차단**
- 시차임기제 (Staggered Terms): 매년 교체 인원 최소화
- 이사회 정원 축소: 1/(N+1) 진입 허들 직접 상승
- 이사 종류별 정원 분리: 집중투표 효과 종류 내 한정
  - 사례: KT&G 2025 — 대표이사 사장 vs 그 외 이사 별개 조 분리, 72% 통과
- 감사위원회 정원 확대 (3→5인): 분리선출 2명 + 나머지 3명 과반 → 무력화

**1-B) 분리선출 감사위원 증원으로 N 감소**
```
집중투표제 진입 최소 지분율 = 1 / (N + 1)
   N = 집중투표 대상 이사 수
   분리선출 감사위원은 집중투표 대상에서 제외
   
분리선출 증원 → N 감소 → 1/(N+1) 상승

예: 이사 9명 → 진입 허들 10%
   분리선출 2명 추가 → N=7 → 12.5% (2.5%p 상승)
```

**1-C) 선결안건 / 운영 전술**
- 임시주총 활용 + 사임·재선임 패턴으로 사실상 시차임기 구축
- 정관 변경 없이도 임기 시점 조정 가능

### OPM 분류

| 안건 | Layer | 권고 |
|---|---|---|
| 시차임기제 | B1 | REVIEW |
| 이사 정수 축소 | B1 | REVIEW |
| 이사 종류별 정원 분리 | B1 | REVIEW |
| 감사위원회 정원 확대 (3→5) | B1 | REVIEW |
| 분리선출 감사위원 증원 (2명 초과) | B1 | REVIEW |
| 임기 미만료 사임·재선임 | B2 | REVIEW |

## 시나리오 2: 보수/임기 우회

### 법 배경
- 남양유업 대법원 판결: 이사인 주주의 보수한도 의결권 제한 (특별이해관계 적용)
- 이사 충실의무 강화 (1차 개정)

### 우회 전술

**2-A) 보수 우회**
- 임직원 보상 프로그램 연계 안건명 포장 → 특별이해관계 회피 시도
- 등기임원 → 미등기임원 전환 → 주총 보수 승인 자체 회피 + 고액 유지
- 보수 규정 정관 직접 명시 → 매년 주총 결의 불필요화 (단 정관변경 결의에 새 쟁점)
- 판례 이전 보수한도 과도 설정 회사: 이미 한도 높아 인상 없이도 우회

**2-B) 임기 우회**
- 주주제안 후보 직위 임기 단축 정관 (예: 3년 → 1년)
- 임기 "3년" → "3년 이내" 유연화 문구
- 임기 미만료 이사 사임 + 재선임 → 사실상 시차임기제

### OPM 분류

| 안건 | Layer | 권고 |
|---|---|---|
| 등기 → 미등기 전환 + 보수 인상 | B1 | REVIEW |
| 보수 규정 정관 직접 명시 | B1 | REVIEW |
| 임기 단축 정관 (3년 → 1년) | B1 | REVIEW |
| 임직원 보상 연계 보수 안건 (포장 의심) | B2 | REVIEW |
| 임기 "3년 이내" 유연화 | B2 | REVIEW |

## 시나리오 3: 합산 3% 룰·의결권 제한 회피

### 법 배경
- 1·2차 개정: 사내·사외 모든 감사위원에 합산 3% 룰 적용
- 적용: 자산 2조+ 또는 1천억-2조 감사위 설치사
- 시행: 2026-07-23

### 기존 vs 개정 비교

| 구분 | 기존 | 개정 |
|---|---|---|
| 사내이사 감사위원 | 합산 3% | 합산 3% |
| 사외이사 감사위원 | **개별 3%** | **합산 3%** |

### 우회 전술

**3-A) 합산 회피 (최대주주 → 2대주주 이전)**
- 최대주주에서 형식적으로 내려옴 → 특관인 합산 규정 자체 회피
- 실질 지배력은 유지
- 신호: 총회 직전 1·2대 주주 지분 역전 → AGAINST 의심

**3-B) 개별 3% 회피 (PRS/TRS 등)**
- PRS (주가수익스왑) / TRS (총수익스왑) → 형식 지분 분산
- 주식 대차 / 사모펀드 활용
- 자사주 재단·기금·우리사주·비특수관계인 출연 → 의결권 부활

**3-C) 상호주 형성**
- 외부 기업과 주식 맞교환 → 비특수관계인 우호 지분 확보
- 대기업집단: 공정거래법상 상호출자 위반 가능
- 소형 상장사: 가능 범위

### OPM 분류

| 안건/신호 | Layer | 권고 |
|---|---|---|
| 자사주 재단·기금·비특수관계인 출연 | B1 | REVIEW |
| 상호주 형성 (대기업집단) | B1 | REVIEW |
| 상호주 형성 (소형 상장사) | B2 | REVIEW |
| 1·2대 주주 지분 역전 (총회 직전) | C | risk_factors |
| PRS/TRS 비공시 | C | risk_factors |

## 시나리오 4: 자사주 의무소각 회피

### 법 배경
- 3차 개정: 자사주 취득 후 1년 내 소각 원칙
- 합병·분할 시 신주 배정 금지
- 시행: 2026-09-10

### 우회 전술

**4-A) 정관 예외 사유 신설**
- 임직원 보상·신기술 도입·경영상 목적 등 폭넓게 규정
- 정상 범위 (법 인정 예외) vs 우회 (의도적 폭넓은 신설) 구분 필요

**4-B) 자사주 출연 (시나리오 3과 동시 효과)**
- 재단·기금·우리사주·비특수관계인 출연
- 효과: 소각 의무 회피 + 의결권 부활 → **일석이조**
- 단, 재단 출연 시 주당 가치 희석 우려

### OPM 분류

| 안건/신호 | Layer | 권고 |
|---|---|---|
| 자사주 합병·분할 시 신주 배정 | A2 | AGAINST (3차 위반) |
| 정관 자사주 보유 예외 사유 신설 | B1·B2 | REVIEW (범위 분석) |
| 자사주 재단·기금·비특수관계인 출연 | B1 | REVIEW |
| 자사주 우리사주(ESOP) 출연 | B2 | REVIEW |
| 자사주 합병/분할 외 신주 배정 | B2 | REVIEW |
| 자사주 출연 공시 | C | risk_factors |

## 36 항목 종합 분류표

### Layer A — 법령 (강행규정)

#### A1. 법 정합 = FOR (8)
1. 집중투표 배제 조항 삭제 (자산 2조+, 2026-09-10)
2. 집중투표 도입 (모든 상장사, 즉시)
3. 감사위원 분리선출 정확 2명 (자산 2조+, 2026-09-10)
4. 감사위원 합산 3% 룰 적용 (자산 2조+ 또는 1천억-2조 감사위, 2026-07-23)
5. 사외이사 → 독립이사 명칭 변경 (모든 상장사, 2025-07-22)
6. 독립이사 비율 정확 1/3 (모든 상장사, 2026-07-23)
7. 전자주주총회 도입 (일정 규모+, 2027-01-01)
8. 자사주 의무소각 결의 (모든 상장사, 2026-09-10)

#### A2. 법 위반 = AGAINST (5)
1. 집중투표 배제 조항 신설/유지 (자산 2조+, 2026-09-10 후)
2. 감사위원 분리선출 1명 이하 (자산 2조+, 2026-09-10 후)
3. 감사위원 합산 3% 룰 미적용 (자산 2조+ 또는 1천억-2조 감사위, 2026-07-23 후)
4. 독립이사 비율 1/3 미달 (모든 상장사, 2026-07-23 후)
5. 자사주 합병·분할 시 신주 배정 (모든 상장사, 2026-09-10 후)

### Layer B — 법 테두리 안 꼼수 (전부 REVIEW)

#### B1. 강한 의심 — 수치/판례 명확 (10)
1. 시차임기제 (Staggered Terms)
2. 등기 → 미등기 임원 전환 + 보수 인상
3. 보수 규정 정관 직접 명시
4. 임기 단축 정관 (예: 3년 → 1년)
5. 자사주 재단·기금·비특수관계인 출연
6. 상호주 형성 (대기업집단)
7. 이사 정수 축소 정관변경
8. 이사 종류별 정원 분리 (KT&G 案)
9. 감사위원회 정원 확대 (3→5)
10. 분리선출 감사위원 증원 (2명 초과)

#### B2. 약한 의심 — 정당 사유 가능 (9)
1. 임기 "3년" → "3년 이내" 유연화
2. 자사주 정관 예외 사유 신설
3. 임직원 보상 프로그램 연계 보수 안건
4. 임기 미만료 이사 사임·재선임
5. 자사주 우리사주(ESOP) 출연
6. 상호주 형성 (소형 상장사)
7. 독립이사 비율 1/2 이상 (의무 초과)
8. 분리선출 감사위원 증원 (자산 2조 미만)
9. 자사주 합병/분할 외 신주 배정

### Layer C — Ownership 신호 (4)
1. 총회 직전 1·2대 주주 지분 역전 (시나리오 3)
2. PRS/TRS 비공시 (시나리오 3)
3. 5% 보유목적 변경 (분쟁 전조)
4. 자사주 출연 공시 (시나리오 3·4 동시)

## OPM 권고 원칙

| 권고 | 조건 |
|---|---|
| **FOR** | 법 정합 (강행규정 의무 정확 충족) |
| **AGAINST** | 명백한 법 위반만 (강행규정 위반) |
| **REVIEW** | 법 테두리 안 모든 의심 케이스 (B1 + B2) — 사람 판단 필요 |
| **risk_factors** | Ownership/contest monitoring 신호 (정관 안건 X) |

이는 OPM 4 기준에 추가:
> **5. 의무 정확 충족 = FOR / 의무 미달 = AGAINST / 의무 초과 + 우회 의심 = REVIEW**
> AGAINST는 법적 근거 명확한 경우만, REVIEW는 본문/맥락/시점 분석 필요한 경우 → 자동 권고 위험 ↓ 정확성 ↑
